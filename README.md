## 소개

**Bira**는 카게로우 프로젝트의 캐릭터 **Ene**를 모티브로 제작 중인 macOS 전용 AI 유틸리티 오버레이 앱입니다.

**Bira** is a macOS AI utility overlay app in development, inspired by **Ene**, a character from the *Kagerou Project*.

항상 화면 위에 떠있는 오버레이 UI로, 사용자가 설정한 인격체와 대화하며 앱 실행, 웹 검색, 파일 제어 등 다양한 유틸리티 기능을 자연스럽게 수행합니다. 단순한 채팅봇이 아닌, 말을 먼저 걸어오기도 하는 옆에 있는 듯한 AI 비서를 목표로 합니다.

It stays on top of your screen as a floating overlay, letting you interact with a customizable AI persona that can launch apps, search the web, manage files, and more — while also starting conversations on its own. The goal is an AI assistant that feels like someone actually *beside* your life.

## Built on OpenClaw 🦞

이 프로젝트는 [openclaw/openclaw](https://github.com/openclaw/openclaw)를 기반으로 제작되었습니다.

OpenClaw는 MIT 라이선스의 오픈소스 개인 AI 어시스턴트 프레임워크로, Gateway 코어, 멀티 모델 Provider 연결, 로컬 툴 실행 등 핵심 인프라를 제공합니다.
Bira는 OpenClaw의 Gateway 백엔드를 활용하고, 그 위에 macOS 네이티브 오버레이 UI와 Ene 인격체 레이어를 더한 프로젝트입니다.

> *This project is built on top of [openclaw/openclaw](https://github.com/openclaw/openclaw), an open-source personal AI assistant framework (MIT License). Bira uses OpenClaw's Gateway as its backend and adds a macOS native overlay UI with a custom persona layer inspired by Ene.*
